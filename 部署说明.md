# 背水对战平台 — 部署说明

## 仓库
https://github.com/Bade-Gusi/bsp-web

## 项目结构
```
├── bsp/          ← WPF 桌面客户端（.NET 9 WPF）
├── bsp-web/      ← Web 客户端（Next.js 14）
└── BeiShui.Server/
    └── src/
        ├── BeiShui.ApiGateway/   ← Web API 后端（ASP.NET Core 9）
        ├── BeiShui.AntiCheat/    ← 反作弊独立服务
        └── BeiShui.MatchService/  ← 匹配服务
```

## 后端部署（BeiShui.ApiGateway）

### 环境要求
- .NET 9.0 SDK
- MySQL 8.0+
- Redis（可选，用于匹配队列）

### 数据库配置
修改 `appsettings.json` 中的连接字符串：
```json
{
  "ConnectionStrings": {
    "DefaultConnection": "server=localhost;port=3306;database=beishui;user=root;password=yourpassword"
  },
  "Jwt": {
    "Key": "改成随机密钥至少32位",
    "Issuer": "BeiShui",
    "Audience": "BeiShuiClient"
  },
  "AntiCheat": {
    "Secret": "改成随机密钥"
  }
}
```

### 启动
```bash
cd BeiShui.Server/src/BeiShui.ApiGateway
dotnet run
# 默认 http://localhost:5000
```

### SignalR Hubs（4个）
```
/hubs/match       → 匹配队列
/hubs/chat        → 聊天
/callhub          → 语音通话/屏幕分享
/hubs/broadcast   → 管理员广播
```

## Web 端部署

```bash
cd bsp-web
npm install
npm run build     # 构建静态文件 → out/
npm start         # 生产模式 :3000
```

或者用 `next.config.js` 中的 API 代理转发到后端（端口 5001）。
